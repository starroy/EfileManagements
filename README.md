# Document OCR Application 


## 	Prerequisites:

​			- Windows 10

### Run E-Office App

   To run the App, Python 3.10 must be installed.

   ​	Install required package
   ```bash
   $ pip install -r requirements.txt
   ```

   ​	Run App

   ```bash
   $ py manage.py riunserver.
   ```

   You can go to the landing page using following link

   ​		http://127.0.0.1:8000
